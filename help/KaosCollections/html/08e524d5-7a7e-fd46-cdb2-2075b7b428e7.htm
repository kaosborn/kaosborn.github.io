Remarks
This is a O(log n) operation.
Examples
In the below snippet, both Skip operations perform an order of magnitude faster than their LINQ equivalent.
C#
Copy
var daryk = new RankedMap<int,int>() { {295,-295} };
for (int i = 2; i < 500; i += 2) daryk.Add (i,-i);

foreach (var x in daryk.Keys.Skip (100).SkipWhile (k => k%2==0).Skip (100))
    Console.Write ($"{x} ");

// Output: 494 496 498
